Plugin for CudaText.
In HTML files (any lexer name with "HTML" word), it finds HTML color tokens: 
#rgb #rrggbb, and adds tooltips for them. 
When you move mouse cursor over color tokens, nice colored tooltip appears.
It finds colors: 
- on opening file,
- after text is changed + pause passed (CudaText option "py_change_slow").
