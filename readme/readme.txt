plugin for CudaText.
in HTML files, it finds HTML color tokens: #rgb #rrggbb, and adds tooltips for them. 
when you move mouse cursor over color tokens, nice colored tooltip appears.
it finds colors: 
- on opening file,
- after text is changed + pause passed (CudaText option "py_change_slow").
