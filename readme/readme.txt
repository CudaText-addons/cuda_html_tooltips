plugin for CudaText.
in HTML files, it finds HTML color tokens:
  #rgb
  #rrggbb
  rgb(nn, nn, nn)
  rgba(nn, nn, nn, aa)
and adds floating tooltips for them. when you move mouse cursor over color tokens, nice colored tooltip appears.
it finds color tokens: 
  - on opening file
  - after text is changed + pause passed.
